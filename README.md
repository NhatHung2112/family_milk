# Family Milk Project
Dự án truy xuất nguồn gốc sữa.

## Hướng dẫn chạy:
1. Backend: `cd backend` -> `pip install -r requirements.txt` -> `python main.py`
2. Frontend: `cd frontend` -> `npm install` -> `npm run dev`
3. Blockchain: `cd blockchain` -> `npm install` -> `npx hardhat run scripts/deploy.js --network ganache`
