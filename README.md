# Family Milk Project
Dự án truy xuất nguồn gốc sữa.

## Hướng dẫn chạy:
1. Backend: `cd backend` -> `npm install` -> `npm start`
2. Frontend: `cd frontend` -> `npm install` -> `npm run dev`
3. Blockchain: `cd blockchain` -> `npm install` -> `npx hardhat run scripts/deploy.js --network ganache`

Tóm tắt thứ tự chạy:
1.Bật MongoDB và Ganache.
2.Chạy Blockchain deploy để có Smart Contract.
3.Chạy Backend 
4.Chạy Frontend.
